# ACAMICA
Entregas académicas de Ciencia de Datos. 

"Proyecto 02": Entrenamiento de modelo de predicción de precios de propiedades en AMBA basado en dataset de la empresa inmobiliaria Properati.

"Proyecto 03": Entrenamiento de modelo de recomendación de juegos de la plataforma Steam basado en reviews de usuarios.

"Conversión de texto a recomendaciones v1": Notebook jupyter escrita en python 3 que puede usarse como script para mejorar el accuracy de un modelo de machine learning de Filtro Colaborativo para un sistema de recomendación de juegos, con métodos de NPL y fuentes de data complementarias mediante web scrapping.
